# gamified-dsa-platform

A gamified DSA learning platform with quests, XP, and leveling.

## Features
- Quest-based learning
- XP system
- Level progression
- Multi-user support

## Tech Stack
- Flask
- MySQL
- HTML/CSS/JS

## Setup

1. Clone repo
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py
